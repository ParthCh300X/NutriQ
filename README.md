# NutriQ

NutriQ is a real-time food intelligence app that scans barcodes and evaluates health risk.

## Features
- 📸 Barcode scanning (CameraX + ML Kit)
- 🧠 Risk analysis engine (Safe / Moderate / Risky / Unknown)
- 📊 Ingredient-based classification
- 🗂️ History tracking (Room DB)

## Tech Stack
- Kotlin
- Jetpack Compose
- CameraX
- ML Kit
- Retrofit
- Room DB
- Hilt

## Vision
Transform food choices into real-time preventive health decisions.
